# Yapılacaklar Listesi 📝

HTML, CSS ve JavaScript ile geliştirilmiş profesyonel bir görev yönetim uygulaması.

---

## 🖥️ Önizleme

> Uygulamayı indirip tarayıcıda açarak kullanabilirsiniz.

---

## ✨ Özellikler

- ✅ Görev ekleme ve silme
- 🔴 Öncelik seviyesi (Yüksek / Orta / Düşük)
- 📅 Son tarih ekleme — geçmiş tarihler kırmızı, yakın tarihler sarı uyarı verir
- 🔀 Sürükle-bırak ile sıralama
- 🔽 Eklenme, öncelik veya tarihe göre sıralama seçeneği
- 🎯 Hepsi / Aktif / Tamamlandı filtresi (animasyonlu geçiş)
- 💾 Otomatik kaydetme — sayfa kapansa bile görevler silinmez
- 📊 Üst özet kartlar — önceliğe göre görev sayısı

---

## 🚀 Kullanım

1. `todo-app.html` dosyasını indirin
2. Herhangi bir tarayıcıda açın
3. Kullanmaya başlayın — kurulum gerekmez!
